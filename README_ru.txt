Tamagotchi
Tamagotchi - это программа на Python, которая моделирует виртуального питомца. Пользователь может взаимодействовать с питомцем, кормить его, лечить и играть с ним. Программа включает три вида питомцев: Кошка, Собака и Попугай.

Установка
Клонируйте репозиторий на свой компьютер:

bash
Copy code
git clone https://github.com/ваш_пользователь/tamagotchi.git
Использование
Откройте терминал или командную строку и перейдите в директорию проекта.

Запустите программу с помощью следующей команды:


python tamagotchi.py

Выберите вид питомца (кошка, собака или попугай), введя соответствующее название.

При запросе введите имя для вашего питомца.

Взаимодействуйте с питомцем, выбирая действия (кормить, лечить, играть), когда вас попросят.

Продолжайте взаимодействие с питомцем до тех пор, пока его счастье не достигнет нуля, или голод не достигнет 100, или здоровье не станет равным нулю.

Игра завершается, и выводится окончательное состояние питомца.
