Tamagotchi
Tamagotchi is a Python program that simulates a virtual pet. Users can interact with the pet by feeding, healing, and playing with it. The program includes three pet species: Cat, Dog, and Parrot.

Installation
Clone the repository to your local machine:



git clone https://github.com/your_username/tamagotchi.git
Usage
Open a terminal or command prompt and navigate to the project directory.

Run the program using the following command:

Copy code
python tamagotchi.py
Choose a pet species (cat, dog, or parrot) by entering the corresponding name.

When prompted, enter a name for your pet.

Interact with the pet by selecting actions (feed, heal, play) when prompted.

Continue interacting with the pet until its happiness reaches zero, or hunger reaches 100, or health becomes zero.

The game ends, and the final state of the pet is displayed.
